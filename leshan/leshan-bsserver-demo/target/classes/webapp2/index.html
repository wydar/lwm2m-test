If you are in development phase and run leshan-bsserver-demo from your ide, you probably missed to launch yarn serve and/or you are not using the right port to access to the demo, try http://localhost:8088/
(see leshan-bsserver-demo README for more details)

If you are using a built version of leshan-bsserver-demo and you see this page, you probably discover a leshan built issue. Please report it.
